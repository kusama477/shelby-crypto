# Shelby Crypto Dashboard

A modern crypto analytics dashboard built for the Shelby ecosystem.

## Overview

Shelby Crypto Dashboard is an open-source analytics platform that provides
real-time cryptocurrency market insights including price trends, trading
volume and market capitalization.

## Features

- Real-time crypto prices
- Interactive price charts
- Market cap rankings
- Portfolio tracking
- Responsive dashboard UI

## Tech Stack

- React
- TailwindCSS
- Chart.js
- CoinGecko API
